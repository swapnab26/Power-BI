# 📊 Swiggy Sales Analytics Dashboard (Microsoft Fabric + Power BI)

## 🔍 Project Overview

This project presents an end-to-end data analytics solution built using **Microsoft Fabric** and **Power BI**, focused on analyzing Swiggy sales data. The goal is to transform raw data into meaningful business insights through a modern data architecture pipeline and interactive dashboard.

The solution demonstrates the complete lifecycle of data — from ingestion and transformation to semantic modeling and visualization.

---

## 🏗️ Architecture & Workflow
1. **Workspace Setup**
   Created a dedicated workspace in Microsoft Fabric to manage all data assets and pipelines.

2. **Data Ingestion (Lakehouse)**
   * Imported raw Swiggy datasets into the Lakehouse
   * Used Lakehouse as the centralized data storage layer

3. **Data Warehousing**
   * Ingested structured data from Lakehouse into the Data Warehouse
   * Enabled optimized querying and analytical processing

4. **Data Transformation (Pipelines)**
   * Built data pipelines to clean, transform, and prepare data
   * Handled missing values, formatting, and schema consistency

5. **Semantic Modeling**
   * Created a semantic model to define relationships, measures, and KPIs
   * Enabled efficient reporting and business logic abstraction

6. **Power BI Integration**
   * Connected Power BI Desktop to the Lakehouse via OneLake catalog
   * Designed interactive dashboards with filters, KPIs, and visuals
